# FP&A Foundations

## Purpose
This repository documents my FP&A foundations and professional development, focusing on how financial planning and analysis supports business decision-making.

## What FP&A Does
FP&A supports leadership by translating financial data into forward-looking insights through forecasting, budgeting, variance analysis, and scenario planning.

## Core FP&A Principles
- Driver-based forecasting over trend-based projections
- Focus on decision impact, not just accuracy
- Clear communication to senior leadership
- Strong linkage between financials and operations

## Tools Used
- Excel (advanced modeling and automation)
- Power BI (dashboarding and KPI reporting)
- Tableau (executive visualization)

## Planned Additions
- Driver-based revenue forecast model
- Budget vs Actual variance analysis
- Executive-style summaries
